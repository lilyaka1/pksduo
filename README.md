# Система складского учета

Практическое задание по дисциплине "Программирование корпоративных систем"

## Описание

Программное обеспечение для автоматизированного управления складской системой с поддержкой логистической оптимизации и распределения товаров по типам хранения.

## Архитектура

### Структура классов

| Класс | Описание | Файл |
|-------|----------|------|
| Product | Товар с характеристиками и сроком годности | Product.cs |
| Warehouse | Склад с типом хранения и массивом товаров | Warehouse.cs |
| WarehouseSystem | Система управления складами и логистикой | WarehouseSystem.cs |
| Program | Точка входа и пользовательский интерфейс | Program.cs |

### Класс Product

**Поля:**
- Id (int) - уникальный идентификатор товара
- SupplierId (int) - идентификатор поставщика
- Name (string) - наименование товара
- VolumePerUnit (double) - объем единицы товара в кубических метрах
- PricePerUnit (double) - цена единицы товара
- DaysUntilExpiry (int) - количество дней до истечения срока годности

**Методы:**
- EditProduct() - редактирование параметров товара
- PrintInfo() - вывод информации о товаре

### Класс Warehouse

**Поля:**
- Id (int) - идентификатор склада
- Type (string) - тип склада (холодный, сортировочный, общий, утилизация)
- Volume (double) - общий объем склада
- Address (string) - адрес расположения
- Products (List<Product>) - список товаров на складе

**Методы:**
- EditWarehouse() - редактирование параметров склада
- PrintInfo() - вывод информации о складе
- GetOccupiedVolume() - расчет занятого объема
- GetFreeVolume() - расчет свободного объема
- AddProduct() - добавление товара
- RemoveProduct() - удаление товара
- CalculateTotalValue() - расчет стоимости товаров

### Класс WarehouseSystem

**Методы:**

**DeliverProducts(List<Product>)** - автоматическая поставка товаров с оптимизацией
- Товары со сроком годности >= 30 дней направляются на общий склад
- Товары со сроком годности < 30 дней направляются на холодный склад
- Товары со смешанными сроками направляются на сортировочный склад

**OptimizeFromSortingWarehouses(int?)** - оптимизация сортировочных складов
- Анализ товаров на сортировочных складах
- Автоматическое распределение по целевым складам согласно срокам годности

**MoveProducts(int, int, List<Product>)** - ручное перемещение товаров
- Перемещение указанных товаров между выбранными складами

**MoveExpiredProducts(int?)** - обработка просроченных товаров
- Автоматический поиск товаров с истекшим сроком годности (дни <= 0)
- Перемещение на склад утилизации

**AnalyzeWarehouseNetwork()** - анализ складской сети
- Выявление нарушений в размещении товаров
- Формирование рекомендаций по оптимизации

**CalculateWarehouseValue(int)** - расчет стоимости товаров на складе

## Функциональность

Программа предоставляет интерактивное меню для управления складами:

1. Просмотр состояния всех складов
2. Создание поставки товаров с автоматическим распределением
3. Оптимизация сортировочных складов
4. Перемещение просроченных товаров на утилизацию
5. Анализ складской сети с выявлением нарушений
6. Расчет стоимости товаров по складам
7. Ручное перемещение товаров между складами

Все операции перемещения товаров сопровождаются логированием в формате:
```
[ЛОГ] Товар 'Название' (объем: X м³) → откуда → куда
```
