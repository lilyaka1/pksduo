# Система складского учета

Практическое задание по дисциплине "Программирование корпоративных систем"

## Быстрый старт

```bash
dotnet run --project WarehouseManagement.csproj
```

## Описание

Программное обеспечение для автоматизированного управления складской системой с поддержкой логистической оптимизации и распределения товаров по типам хранения.

## Функционал

1. Просмотр складов - показывает состояние всех складов
2. Просмотр товаров - список товаров на каждом складе
3. Добавить поставку - ввод новых товаров с автоматической оптимизацией
4. Удалить товар - выбор склада и товара для удаления
5. Оптимизация сортировочных складов
6. Перемещение просроченных товаров
7. Анализ складской сети
8. Расчет стоимости товаров
